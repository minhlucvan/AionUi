# Interface Design Workspace

A workspace for building crafted, consistent user interfaces with design memory and enforcement.

## Structure

```
.interface-design/     # Design system storage
  system.md            # Your design decisions and tokens
.claude/
  commands/            # Plugin commands (init, status, audit, extract, critique)
  skills/              # Plugin skills and references
reference/             # Design templates and examples
```

## Usage

Start by running the init command to establish your design direction, then build components that follow your system consistently.
