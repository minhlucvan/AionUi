# 3D Game Development Assistant

A minimal Claude Code workspace for 3D game development with Three.js, providing AI-powered assistance through specialized skills and expert agents.

## What's Included

This workspace provides Claude Code with specialized knowledge for 3D game development:

### Skills

- **threejs-scene-setup** - Scene, camera, renderer configuration best practices
- **game-physics** - Cannon.js integration and collision handling
- **shader-programming** - Custom GLSL shader development
- **performance-optimization** - FPS monitoring, instancing, LOD techniques

### Agents

- **game-developer** - Main agent: senior 3D game developer and architect. Use `@game-developer` as the primary persona for all game development conversations. Delegates to specialist agents when needed.
- **graphics-expert** - Specialist: advanced rendering techniques and shader optimization
- **physics-engineer** - Specialist: complex physics simulations and collision systems

## Usage

This workspace is automatically loaded when you start a conversation with the 3D Game Development assistant. Claude will have access to all skills and agents defined here.
