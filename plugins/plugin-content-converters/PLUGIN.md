# Content Converters Plugin

Plugin for converting documents between different formats.

## Features

### PDF to PowerPoint
- Convert PDF documents to editable PowerPoint presentations
- Handle watermark removal during conversion
- Optimize layouts for slide format
- Extract charts and images from PDFs
