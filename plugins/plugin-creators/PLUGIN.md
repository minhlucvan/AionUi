# Creators Plugin

Plugin for generating creative interactive content.

## Features

### 3D Game Creator
- Generate complete 3D platform collection games
- Single HTML file output (standalone, no dependencies)
- Three.js-based 3D graphics
- Physics and collision detection
- Player controls (WASD + Jump)
- Collectibles and scoring system
